# Todo Plugin for KOReader

A lightweight local todo list plugin for KOReader.

## Features

- Works in both **File Browser** and **Reader**.
- Top-left `+` button to create a new todo.
- Tap an item to toggle done/undone.
- Long-press an item to edit or delete it.
- Optional content field per todo.
- Main menu toggle: **Hide completed items**.
- Main menu option: **Sort order** (`Ascending` / `Descending`).
- Gesture action available in Gesture Manager:
  - `Todo: Open list` (`todo_open_list`)

## Notes

- Works normally on Kindle Paperwhite 11th gen (PW5).
